## Project Overview

This is a Python project for Slack data export processing, currently in initial setup phase.

## Environment Setup

- Python virtual environment is located in `.venv/`
- Python version: 3.14
- Activate virtual environment: `source .venv/bin/activate` (Unix/macOS) or `.venv/Scripts/activate` (Windows)

## Development Workflow

When adding code to this project:
- Install dependencies using: `pip install <package>`
- Freeze dependencies to requirements.txt: `pip freeze > requirements.txt`
- Run Python scripts from the activated virtual environment
